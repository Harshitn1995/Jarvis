⚠️ Integrate Jarvis with Your App or Website Before You Begin

Before using any features, make sure the Jarvis SDK is properly integrated with your app or website. This allows the platform to start capturing critical engagement data:

  • System User Attributes, System Events, and System Event Attributes
  • Custom User Attributes, Custom Events, and Custom Event Attributes tailored to your business
